# ☕︎ Santander Imersão Digital - Trilha Mobile (Alura)

Este repositório reúne os estudos e projetos desenvolvidos durante a **Santander Imersão Digital**, em parceria com a **Alura**, na **trilha Mobile**. O programa teve como objetivos introduzir conceitos e práticas fundamentais do **JavaScript** até o **Dart**.

## ☕︎ Objetivo da Trilha
Aprender a criar aplicativos Android do zero, explorando desde conceitos básicos até a construção de interfaces e interação com o usuário, aplicando boas práticas de desenvolvimento mobile.

## ☕︎ Conteúdos Estudados 
- Semana 1
  - Começando em Programação: carreira e primeiros passos
  - Lógica de Programação: mergulhe em programação com JavaScript
- Semana 2
  - Lógica de Programação: explore funções e listas
- Semana 3
  - Git e GitHub: compartilhando e colaborando em projetos
- Semana 4
  - Lógica de Programação: praticando com desafios

## ☕︎ Tecnologias e Ferramentas
- JavaScript
- VS Code
